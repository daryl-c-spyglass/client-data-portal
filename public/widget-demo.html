Spyglass Property Search Widget

Embeddable property search for WordPress

WordPress Integration

To add this widget to your WordPress site, add the following code:

<!-- Add this where you want the search widget to appear -->
<div id="spyglass-search"></div>

<!-- Add this before the closing </body> tag -->
<script src="https://your-replit-url.replit.app/spyglass-property-widget.js"></script>
<script>
  SpyglassPropertySearch.init('spyglass-search', {
    apiUrl: 'https://your-replit-url.replit.app/api/widget',
    googleMapsKey: 'YOUR_GOOGLE_MAPS_API_KEY' // Optional, for map view
  });
</script>
