### DIAGNOSE

**1. Find the map container in PropertyDetail**

Search for the Location section and the MapboxMap component usage.

**2. Check for height issues**

Mapbox requires an explicit height on the container. If the container has `height: 0` or no height set, the map won't be visible even though it's initialized.

Look for:
```tsx
<MapboxMap 
  // ... props
  className="..."  // Does this have height?
  style={{ height: ??? }}  // Is height explicitly set?
/>
```

**3. Common causes:**

- Container has `height: auto` or no height (map needs explicit px or viewport height)
- Parent container has `overflow: hidden` with 0 height
- Map container has `display: none` or `visibility: hidden`
- Flexbox/grid parent not giving the map space

---

### LIKELY FIX

Ensure the map container has an explicit height:
```tsx
// In PropertyDetail.tsx, Location section:
<div className="h-[400px] w-full">  {/* Explicit height required */}
  <MapboxMap
    latitude={property.latitude}
    longitude={property.longitude}
    // ... other props
    className="h-full w-full rounded-lg"
  />
</div>
```

Or with inline style:
```tsx
<MapboxMap
  // ... props
  style={{ height: '400px', width: '100%' }}
/>
```
