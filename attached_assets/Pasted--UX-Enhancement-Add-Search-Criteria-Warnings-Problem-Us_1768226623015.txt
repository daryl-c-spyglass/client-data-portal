# UX Enhancement: Add Search Criteria Warnings

## Problem
Users may not realize why they're getting unexpected results:
1. No subdivision = all subdivisions in ZIP appear
2. Narrow sqft + short date range = 0 Closed results

## Task: Add Warning Messages

### Warning 1: No Subdivision Filter
When subdivision is empty but ZIP is filled:
```jsx
{!criteria.subdivision && criteria.zipCode && (
  <div className="bg-yellow-50 border border-yellow-200 rounded-lg p-3 mb-4 text-sm">
    <span className="font-medium text-yellow-800">ℹ️ Tip:</span>
    <span className="text-yellow-700 ml-1">
      No subdivision filter applied. Results include ALL subdivisions in ZIP {criteria.zipCode}.
      <button 
        onClick={() => focusSubdivisionField()}
        className="text-yellow-800 underline ml-1"
      >
        Add subdivision filter
      </button>
    </span>
  </div>
)}
```

### Warning 2: Narrow Search with 0 Closed Results
When Closed is selected but returns 0:
```jsx
{criteria.statuses.includes('Closed') && counts.closed === 0 && (
  <div className="bg-blue-50 border border-blue-200 rounded-lg p-3 mb-4 text-sm">
    <span className="font-medium text-blue-800">ℹ️ No Closed Listings Found</span>
    <p className="text-blue-700 mt-1">
      No properties matching your criteria have closed in the last {criteria.closeDateDays} days.
    </p>
    <div className="mt-2 flex gap-2">
      <button 
        onClick={() => updateCloseDateDays(365)}
        className="text-xs px-2 py-1 bg-blue-100 text-blue-800 rounded hover:bg-blue-200"
      >
        Try 365 days
      </button>
      <button 
        onClick={() => clearSqftFilters()}
        className="text-xs px-2 py-1 bg-blue-100 text-blue-800 rounded hover:bg-blue-200"
      >
        Remove sqft limits
      </button>
    </div>
  </div>
)}
```

### Warning 3: Very Narrow Sqft Range
When sqft range is less than 2000:
```jsx
{criteria.minSqft && criteria.maxSqft && 
 (criteria.maxSqft - criteria.minSqft) < 2000 && (
  <div className="text-xs text-amber-600 mt-1">
    ⚠️ Narrow range ({criteria.maxSqft - criteria.minSqft} sqft) may limit results
  </div>
)}
```

## Acceptance Criteria
- [ ] Warning appears when no subdivision is entered
- [ ] Warning appears when Closed count is 0
- [ ] Suggestions are actionable (one-click to fix)
- [ ] Warnings don't block search, just inform
